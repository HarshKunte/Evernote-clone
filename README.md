# Notey
### Notes web app built using React, Material-UI and Firebase along with Google authentication.

## [Check live here](https://mynotey.netlify.app).

## Features
- Create Notes
- Edit Notes
- Bookmark Notes
- Share Notes
- View only Note
- Access notes anywhere using your Google account.
- Mobile responsive
